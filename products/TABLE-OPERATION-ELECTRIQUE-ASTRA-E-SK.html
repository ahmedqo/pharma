Caractéristiques

La table d'opérationA100-4est adaptée aux différents types d’opérations chirurgicales: chirurgie thoracique, abdomen, périnée, membres inférieurs, gynécologie, urologie, ORL, ainsi que neurochirurgie.

Toutes les fonctions sont contrôlées à l'aide d'une télécommande et peuvent être verrouillées.

Toutes les sections disposent d’une cassette à rayon X pour examen radiologique.

La table d'opération est équipée de batterie intégrée haute performance afin de garantir le fonctionnement jusqu’à 80 opérations.
